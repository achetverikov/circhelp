# Data from an orientation estimation task

A dataset with the orientation estimation results from Experiment 2 in
Pascucci et al. (2019) available from
https://doi.org/10.5281/zenodo.2544946.

## Usage

``` r
Pascucci_et_al_2019_data
```

## Format

A data frame with 4400 rows and 5 variables:

- observer:

  observer ID

- orientation:

  true orientation

- reported:

  reported orientation

- rt:

  response time

- err:

  estimation error
